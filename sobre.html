Nossa Missão

O **The Cocktail Bar** nasceu como um projeto acadêmico focado em demonstrar a manipulação de dados JSON, eventos DOM e estilização moderna usando HTML, CSS e JavaScript.

Nosso objetivo é fornecer uma biblioteca elegante e funcional de receitas de bebidas e coquetéis, permitindo que o usuário encontre rapidamente a receita desejada através de filtros e buscas interativas.

Tecnologias Utilizadas

  • **HTML5:** Estruturação semântica do conteúdo.
  • **Tailwind CSS:** Estilização utilitária para um design rápido e elegante.
  • **JavaScript ES6+:** Lógica de carregamento de dados e interatividade (filtros e busca).
  • **JSON:** Formato de dados para o catálogo de coquetéis.
